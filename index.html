A service worker act as proxy between your web application, the browser and the network

  • Push notifications

ðŸ“– Service Worker API

Uses recent async APIs

support: chrome, mozilla, android, opera, edge, safari

  • Promises construction to executes code that will resolve later
  • postMessage method to exchange messages and objects between contexts
  • fetch function to make simple HTTP request, successor of XHR

Offline basics

support: chrome, mozilla, android, opera, edge (developement), safari (developement)

service-worker.js

const CACHE = 'cache-v1',
const FILES = ['/', '/styles.css', '/script.js']
self.addEventListener('install', event =>
    event.waitUntil(
        caches.open(CACHE)
            .then(cache => cache.addAll(FILES))
    )
)
